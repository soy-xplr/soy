# 디자인 방향

## 한 줄 컨셉

좋아서 모아둔 인터넷 조각을 손글씨 노트에 붙여둔 공개 스크랩북.

## 무드 키워드

- 따뜻한
- 개인적인
- 정돈된
- 손으로 붙인 듯한
- 오래 다시 펼쳐보고 싶은

## 레이아웃 원칙

- 전체 콘텐츠 폭은 너무 넓히지 않고 노트 한 페이지처럼 중앙에 모은다.
- 상단에는 큰 제목, 짧은 소개, 검색과 필터를 순서대로 배치한다.
- 링크 목록은 카드 그리드로 구성한다.
- 세부 페이지는 노션의 일반 페이지처럼 좁은 본문 폭과 세로형 콘텐츠 흐름을 사용한다.
- 섹션 사이에는 얇은 구분선이나 넉넉한 여백을 사용한다.
- 장식은 페이지 가장자리나 카드의 작은 포인트에만 배치한다.

## 컬러 팔레트

| 역할 | 색상 | 사용처 |
| --- | --- | --- |
| 종이 배경 | `#FAF7F1` | 페이지 전체 배경 |
| 카드 배경 | `#FFFDF8` | 링크 카드 |
| 잉크 텍스트 | `#2F2F2F` | 본문과 제목 |
| 보조 텍스트 | `#6F6A60` | 설명, 날짜, URL |
| 구분선 | `#E3DDD2` | 섹션 라인, 카드 테두리 |
| 형광펜 노랑 | `#F6D76B` | 제목 밑줄, 추천 표시 |
| 스티커 블루 | `#B7D6F2` | 태그, 필터 포인트 |
| 메모 핑크 | `#F4B6C2` | 강조 라벨 |

## 타이포그래피

- 제목: 손글씨 느낌의 폰트를 사용하되, 가독성이 높은 것을 선택한다.
- 본문: 산세리프 폰트로 읽기 편하게 유지한다.
- 링크 카드 제목: 손글씨체를 남용하지 않고 본문 폰트의 굵기 변화로 정리한다.
- 손글씨 폰트는 로고, 짧은 메모, 장식성 라벨에 제한적으로 사용한다.

## 컴포넌트 스타일

### 히어로

- 제목 아래에 형광펜 밑줄을 둔다.
- 소개 문구는 1-2줄로 짧게 작성한다.
- 오른쪽 위나 아래에 작은 손그림 화살표 또는 별표를 배치할 수 있다.

### 검색창

- 종이에 연필로 그린 듯한 얇은 테두리를 사용한다.
- placeholder는 부드러운 문장으로 작성한다.
- 예시: `찾고 싶은 링크 조각을 적어보세요`

### 카테고리 필터

- 알약 형태 또는 작은 스티커 형태로 표현한다.
- 선택 상태는 배경색과 살짝 두꺼운 테두리로 구분한다.
- 전체 보기 버튼은 항상 첫 번째에 둔다.

### 링크 카드

- 카드 배경은 종이보다 살짝 밝게 둔다.
- 카드별로 미세한 회전 값을 줄 수 있지만, 읽기 어려울 정도로 기울이지 않는다.
- 썸네일은 테이프로 붙인 사진처럼 보이게 할 수 있다.
- 개인 메모는 카드 안의 작은 메모 조각처럼 분리한다.
- 카드 전체가 세부 페이지로 이어지는 느낌을 주되, 클릭 가능하다는 점이 명확해야 한다.

### 세부 페이지

- 노션의 일반 페이지처럼 본문 폭을 좁게 유지하고 위에서 아래로 자연스럽게 읽히게 한다.
- 상단에는 제목, 카테고리, 태그, 저장일을 배치한다.
- 커버 이미지나 대표 이미지는 큰 카드처럼 보여주고, 필요하면 테이프 장식을 더한다.
- 본문에는 텍스트 문단, 이미지, 짧은 메모 블록을 하드코딩으로 배치한다.
- 원본 링크는 본문 상단 또는 하단에 작게 제공한다.
- 뒤로 가기 또는 목록으로 돌아가기 링크를 페이지 상단에 둔다.

### 태그

- 태그는 작은 스티커처럼 보이게 한다.
- 태그 수가 많아도 카드가 복잡해지지 않도록 3-4개 안팎을 권장한다.

### 빈 상태

- 차가운 오류 문구 대신 노트 톤의 문장을 사용한다.
- 예시: `아직 이 서랍에는 아무것도 없어요.`

## 이미지 스타일

- 썸네일 비율은 카드 안에서 일관되게 유지한다.
- 이미지가 없을 때는 단색 종이 조각, 작은 아이콘, 카테고리 라벨로 대체한다.
- 이미지에는 너무 강한 필터를 적용하지 않는다.

## 인터랙션 원칙

- 카드 hover 시 살짝 떠오르거나 기울기가 완만하게 바뀐다.
- 카드 클릭 시 내부 세부 페이지로 이동한다.
- 세부 페이지의 원본 링크는 새 탭으로 열린다.
- 필터 선택은 즉시 반영된다.
- 검색 입력은 빠르게 반응해야 한다.
- 장식 애니메이션은 짧고 조용하게 유지한다.

## 카피 톤

문장은 짧고 개인적인 기록처럼 쓴다. 기능 설명도 명령형보다 초대하는 말투를 사용한다.

권장 문구:

- 좋아서 저장해둔 인터넷의 작은 조각들
- 오늘은 어떤 링크를 펼쳐볼까요?
- 다시 읽고 싶은 페이지
- 한동안 곁에 두고 싶은 도구
- 원본 링크 펼쳐보기
- 목록으로 돌아가기
- 아직 이 서랍에는 아무것도 없어요

피할 문구:

- 데이터가 없습니다
- 검색 결과 없음
- 링크 목록
- 카테고리 필터링 완료

## 접근성 메모

- 손글씨 폰트는 장식적으로만 사용하고 핵심 정보에는 가독성 높은 폰트를 사용한다.
- 텍스트와 배경의 대비를 충분히 확보한다.
- 필터, 카드, 세부 페이지의 원본 링크 액션은 키보드로 접근 가능해야 한다.
- 색상만으로 선택 상태를 전달하지 않고 테두리, 굵기, 라벨도 함께 사용한다.
