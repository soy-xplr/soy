# MVP 범위

## 목표

첫 버전의 목표는 방문자가 감성적인 링크 큐레이션 사이트를 둘러보고, 관심 있는 링크를 검색하거나 카테고리로 좁혀 내부 세부 페이지로 이동할 수 있게 하는 것이다. 세부 페이지에서는 사이트 소유자가 직접 하드코딩한 텍스트와 이미지를 노션의 일반 페이지처럼 읽을 수 있다.

## 포함 기능

### 1. 홈 소개 영역

- 사이트 제목을 손글씨 스타일로 보여준다.
- 사이트의 성격을 설명하는 짧은 소개 문구를 제공한다.
- 추천 링크가 있다면 상단에서 시각적으로 강조한다.

### 2. 링크 카드 목록

- 정적 링크 데이터를 카드 그리드로 표시한다.
- 각 카드는 제목, 설명, 개인 메모, 카테고리, 태그, 썸네일, 저장일을 보여준다.
- 카드의 주요 액션은 내부 세부 페이지로 이동하는 것이다.

### 3. 북마크 세부 페이지

- 각 북마크는 고유한 세부 페이지를 가진다.
- 세부 페이지는 노션의 일반 페이지처럼 제목, 이미지, 본문 텍스트를 세로로 배치한다.
- 세부 페이지 내용은 사이트 소유자가 코드에 직접 하드코딩한다.
- 원본 외부 링크는 세부 페이지 안에서 새 탭으로 열 수 있게 제공한다.
- 홈 또는 목록으로 돌아가는 링크를 제공한다.

### 4. 카테고리 필터

- 기본값은 전체 보기다.
- 카테고리를 선택하면 해당 카테고리의 링크만 보여준다.
- 선택된 카테고리는 시각적으로 분명하게 표시한다.

### 5. 키워드 검색

- 검색 대상은 제목, 설명, 개인 메모, 카테고리, 태그다.
- 검색어와 카테고리 필터는 함께 적용될 수 있다.
- 검색 결과가 없을 때는 빈 상태 메시지를 보여준다.

### 6. 반응형 레이아웃

- 모바일에서는 카드가 1열로 표시된다.
- 태블릿과 데스크톱에서는 2-3열 카드 그리드를 사용한다.
- 검색창과 필터는 좁은 화면에서도 쉽게 누를 수 있어야 한다.

## 제외 기능

- 브라우저에서 링크를 추가, 수정, 삭제하는 관리 화면
- URL에서 제목, 설명, 이미지 자동 수집
- 세부 페이지 본문을 CMS, 데이터베이스, 마크다운으로 관리하는 기능
- 로그인과 사용자별 저장소
- 댓글, 좋아요, 북마크 저장 같은 소셜 기능
- 서버 데이터베이스나 API
- 링크 클릭 통계
- 다국어 지원

## 완료 기준

- 샘플 링크 데이터 8개 이상을 화면에 표시할 수 있다.
- 각 북마크 카드에서 대응되는 내부 세부 페이지로 이동할 수 있다.
- 세부 페이지에는 하드코딩된 텍스트와 이미지 영역이 있다.
- 방문자가 카테고리 필터로 링크 목록을 좁힐 수 있다.
- 방문자가 키워드 검색으로 링크를 찾을 수 있다.
- 세부 페이지 안의 외부 링크가 새 탭으로 열린다.
- 검색 결과가 없을 때 빈 상태가 표시된다.
- 모바일과 데스크톱에서 주요 콘텐츠가 읽기 좋게 보인다.

## 우선순위

| 우선순위 | 항목 | 이유 |
| --- | --- | --- |
| P0 | 링크 카드 목록 | 제품의 핵심 경험 |
| P0 | 북마크 세부 페이지 | 카드에서 더 깊은 큐레이션 내용을 읽는 핵심 흐름 |
| P0 | 카테고리 필터 | 공개 큐레이션 탐색에 필요 |
| P0 | 키워드 검색 | 링크 수가 늘어날 때 실용성 확보 |
| P1 | 추천 링크 강조 | 사이트 소유자의 취향을 빠르게 전달 |
| P1 | 빈 상태 | 탐색 중 막힘을 줄임 |
| P2 | 카드별 미세한 회전/장식 | 감성은 살리되 기능 이후 적용 |
