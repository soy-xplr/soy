# 상세 페이지 하위 작업 카드 섹션 구상

## 목적

헬로메이플처럼 하나의 큰 프로젝트 안에 여러 담당 범위가 있는 경우, 상세 페이지를 길게 이어 쓰는 대신 하위 작업을 카드 단위로 정리한다.

## 기본 구조

상세 페이지는 다음 순서로 구성한다.

1. 프로젝트 개요
2. 핵심 성과
3. 담당 범위 카드 섹션
4. 카드별 상세 내용

## 하위 작업 카드 예시

헬로메이플 상세 페이지에서는 다음 카드가 들어갈 수 있다.

- 서비스 런칭 PM
- 회원 시스템 및 계정 정책
- 공식 홈페이지 리뉴얼
- 백오피스 및 운영툴 구축
- 글로벌 서비스 확장
- 데이터 기반 개선

## 카드에 표시할 정보

각 카드는 너무 길지 않게 아래 정보만 먼저 보여준다.

- 하위 작업명
- 한 줄 요약
- 주요 역할 태그
- 기간 또는 오픈 시점
- 대표 이미지 또는 아이콘

## 카드 클릭/펼침 후 상세 정보

카드를 클릭하거나 펼치면 다음 구조를 보여준다.

- 문제 상황
- 내가 한 일
- 협업 대상
- 결과
- 이미지/문서 캡처

## 편집 모드 초안

소유자 편집 모드에는 `하위 작업 카드 추가` 버튼을 둔다.

카드 편집 필드는 다음과 같다.

- 카드 제목
- 요약
- 역할 태그
- 기간
- 대표 이미지 URL/파일
- 문제 상황
- 내가 한 일
- 결과

## 구현 전 결정할 것

- 카드를 클릭하면 펼침형으로 보여줄지, 상세 페이지 내 앵커로 이동할지
- 하위 작업마다 여러 이미지를 허용할지
- 카드 순서를 드래그로 바꿀지, 위/아래 버튼으로 바꿀지
- 기존 일반 섹션과 하위 작업 카드를 함께 쓸지, 큰 프로젝트에는 카드 섹션만 쓸지
